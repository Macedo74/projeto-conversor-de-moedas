# 💸 DevClub - Convert Money

Um conversor de moedas moderno, intuitivo e totalmente responsivo, desenvolvido como parte do currículo do DevClub. O projeto permite converter valores entre Real, Dólar, Euro, Libra e Bitcoin, com atualização visual dinâmica de bandeiras e nomes.

---

## 🚀 Demonstração

O projeto foi construído focado na experiência do usuário (UX), garantindo que a interface seja amigável tanto em computadores quanto em dispositivos móveis.



## ✨ Funcionalidades

- **Conversão Bidirecional:** Converta de qualquer moeda para qualquer outra disponível na lista.
- **Formatação Automática:** Utiliza a API `Intl.NumberFormat` para exibir os valores com os símbolos e casas decimais corretos de cada país/moeda.
- **Interface Dinâmica:** As bandeiras e nomes das moedas mudam automaticamente ao selecionar uma opção.
- **Totalmente Responsivo:** Design que se adapta perfeitamente a telas de celulares, tablets e desktops.

## 🛠️ Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as tecnologias fundamentais da web:

- **HTML5:** Estrutura semântica dos elementos.
- **CSS3:** Estilização, Flexbox e Media Queries para responsividade.
- **JavaScript:** Lógica de conversão, manipulação do DOM e formatação de moedas.
- **Google Fonts:** Uso das fontes *Roboto* e *Oswald* para um design moderno.
